# E-Passport System

A full-stack e-passport application system with user portal and admin dashboard.

## Stack
- **Backend**: Node.js + Express + TypeScript + SQLite + JWT
- **Frontend**: React + TypeScript + Vite + Tailwind CSS

## Quick Start

### Backend
```bash
cd backend
npm install
npm run dev
```
Runs on http://localhost:5000

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Runs on http://localhost:3000

## Default Admin Account
- Email: admin@epassport.gov
- Password: Admin@123

## Features
- User registration and login
- Multi-step passport application form
- Document uploads (photo + ID)
- Real-time application status tracking
- Admin dashboard with stats and review workflow
